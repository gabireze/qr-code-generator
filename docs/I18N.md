# Internacionalização (i18n)

## Visão Geral

O QR Code Generator agora suporta **Português Brasileiro (pt-BR)** e **Inglês (en)**.

## Funcionalidades

### Detecção Automática de Idioma
- O sistema detecta automaticamente o idioma do navegador
- Se o navegador estiver em **português** (qualquer variante pt, pt-BR, pt-PT), o site carrega em português brasileiro
- Para **qualquer outro idioma**, o site carrega em inglês

### Seletor de Idioma
- Um botão com ícone de idioma aparece no header
- Permite alternar entre PT e EN manualmente
- A preferência do usuário é salva no `localStorage`

### Persistência
- A escolha manual do idioma é salva localmente
- Na próxima visita, o site carrega no idioma escolhido anteriormente
- Se não houver preferência salva, usa a detecção automática

## Estrutura Técnica

### Arquivo Principal: `/src/js/i18n.js`

Contém:
- Objeto `translations` com todas as traduções
- Classe `I18n` para gerenciamento
- Detecção automática de idioma
- Atualização dinâmica do DOM

### Atributos HTML

#### `data-i18n`
Usado para texto de elementos:
```html
<h1 data-i18n="header.title">QR Code Generator</h1>
```

#### `data-i18n-placeholder`
Usado para placeholders de inputs:
```html
<input data-i18n-placeholder="placeholder.url" placeholder="https://example.com">
```

#### `data-i18n-title`
Usado para títulos (tooltips):
```html
<button data-i18n-title="tooltip.download" title="Download">
```

## Sobre a URL

**A URL permanece a mesma**: `https://qrcode.gabireze.cloud/`

Não é necessário ter URLs diferentes por idioma porque:
1. ✅ Melhor experiência do usuário (uma única URL para compartilhar)
2. ✅ SEO otimizado com `hreflang` tags
3. ✅ Detecção automática funciona perfeitamente
4. ✅ Usuário pode alternar manualmente se preferir

### Meta Tags SEO

O sistema atualiza automaticamente:
- `<title>`
- `<meta name="description">`
- `<meta property="og:*">` (Open Graph)
- `<meta name="twitter:*">` (Twitter Cards)
- `lang` attribute do HTML

## Como Adicionar Novas Traduções

1. Abra `/src/js/i18n.js`
2. Adicione a chave nos objetos `pt-BR` e `en`:

```javascript
'pt-BR': {
  'minha.nova.chave': 'Texto em português',
  // ...
},
'en': {
  'minha.nova.chave': 'Text in English',
  // ...
}
```

3. Use no HTML com `data-i18n`:

```html
<span data-i18n="minha.nova.chave">Texto em português</span>
```

## Como Adicionar um Novo Idioma

1. Adicione um novo objeto de idioma em `translations`:

```javascript
const translations = {
  'pt-BR': { /* ... */ },
  'en': { /* ... */ },
  'es': {  // Espanhol
    'header.title': 'Generador de Código QR',
    // ... todas as outras chaves
  }
};
```

2. Atualize o método `detectLanguage()`:

```javascript
detectLanguage() {
  const browserLang = navigator.language || navigator.userLanguage;
  
  if (browserLang.toLowerCase().startsWith('pt')) {
    return 'pt-BR';
  }
  if (browserLang.toLowerCase().startsWith('es')) {
    return 'es';
  }
  return 'en'; // padrão
}
```

3. Atualize o seletor de idioma no método `createLanguageSelector()`.

## Eventos Customizados

O sistema dispara um evento quando o idioma muda:

```javascript
window.addEventListener('languageChanged', (event) => {
  console.log('Novo idioma:', event.detail.language);
  // Faça algo quando o idioma mudar
});
```

## API do i18n

### Obter tradução
```javascript
i18n.t('header.title') // Retorna "Gerador de QR Code" ou "QR Code Generator"
```

### Obter idioma atual
```javascript
i18n.getLanguage() // Retorna "pt-BR" ou "en"
```

### Definir idioma
```javascript
i18n.setLanguage('en') // Muda para inglês
```

### Atualizar página
```javascript
i18n.updatePage() // Atualiza todo o conteúdo traduzível
```

## Testando

1. **Teste de detecção automática**: 
   - Limpe o localStorage: `localStorage.clear()`
   - Recarregue a página
   - Deve carregar no idioma do navegador

2. **Teste de seletor manual**:
   - Clique no botão de idioma no header
   - Verifique se todo o texto muda
   - Recarregue a página
   - Deve manter o idioma escolhido

3. **Teste em diferentes navegadores**:
   - Chrome: Configurações > Idiomas
   - Firefox: Preferências > Idioma
   - Edge: Configurações > Idiomas
